# tonchtory
TÖNCHTORY é um sistema de inventário offline focado em agilidade e precisão. Permite contagem com ou sem planilha: cada bip cria/atualiza itens, calcula status (OK/SOBRA/FALTA/ZERADO/NOVO) e mantém a consistência da conferência. Interface padronizada, fluxo simples e exportação em XLSX/CSV para uso real em estoque.
